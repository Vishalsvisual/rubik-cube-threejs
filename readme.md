# Rubik Cube

ThreeJs Rubik's Cube game supports the custom Rubik's Cube class (currently only 2-10 Rubik's Cube is opened on the interface)。

## Demo

Trial address：[Rubiks Cube - Cube]()

## Operating project

1. Install

```shell
npm install
```

2. run

```shell
npm run start
```

3. Open the browser

Browser input `http://localhost:8000`。
